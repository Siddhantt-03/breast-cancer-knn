# Breast Cancer Classification using KNN

## 📌 Overview

This project applies the K-Nearest Neighbors (KNN) algorithm to classify breast cancer tumors as malignant or benign.

## ⚙️ Features

* Data preprocessing and cleaning
* Feature scaling using StandardScaler
* Data visualization (heatmap, distribution plots)
* Model training with multiple K values
* Accuracy comparison
* Confusion matrix evaluation

## 📊 Dataset

The dataset contains features computed from breast mass images and a target variable:

* M → Malignant (1)
* B → Benign (0)

## 🚀 How to Run

1. Clone the repository
2. Install dependencies:

   ```bash
   pip install -r requirements.txt
   ```
3. Run the notebook in Jupyter or Google Colab

## 📈 Results

* Model trained with multiple K values
* Best K selected based on accuracy
* Evaluation using confusion matrix

## 🧠 Key Learnings

* Importance of feature scaling in KNN
* Bias-variance tradeoff with K values
* Model evaluation techniques
